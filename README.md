# cupping-app
             Simple in use application for cupping coffee at factory and retail coffee shops.
## Start project:
### before install:
    * npm / node.js / expo (maybe something else too)
### after that
    * git clone   *[ project ]*
    * cd cupping-app
    * npm install
    * npm start

### use iOS simiulator or android emulator to run this app
